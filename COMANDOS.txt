COMANDOS PARA REPOSITORIO

Por primera vez, hay dos formas:
1- Crean carpeta en su compu
2- Abren git bash en esa carpeta
3- ejecutan "git init"
4- ejecutan git remote add origin https://github.com/natitejeda/ProyectoPescar.git
5- ejecutan git pull origin main (para traer lo que ya hay en el repositorio)
